# Политика конфиденциальности PLUGIN.FLP

**Версия:** 1.0 | GDPR-ready для пользователей из ЕС

Оператор: `[НАИМЕНОВАНИЕ]`, `[EMAIL]`

## Какие данные

| Данные | Зачем |
|--------|--------|
| Email, пароль (хэш) | Аккаунт (Supabase Auth) |
| device_id | Привязка ПК, анти-абуз |
| Метаданные генераций (модель, длина промпта, **без текста промпта**) | Квота, биллинг ИИ |
| IP (кратковременно) | Безопасность, rate limit |
| Платежи (через ЮKassa/Stripe) | Подписка; **логи 2+ года** (анонимизируются при удалении аккаунта) |

**Тексты промптов не храним** на сервере дольше обработки запроса.

## Срок хранения

- Метаданные генераций: **90 дней** (настраивается `PLG_PROMPT_LOG_RETENTION_DAYS`).
- Платёжные события: **730+ дней** (налоги, chargeback).
- Аккаунт: до удаления вами.

## Регион данных

БД: Supabase **EU** (`eu-central-1`) для EU/CIS пользователей. При расширении в US — отдельный проект.

## Права (GDPR / 152-ФЗ)

- Доступ, исправление, удаление — **Настройки → Удалить аккаунт** или `[EMAIL]`.
- Удаление стирает PII; платежи анонимизируются.
- Право на жалобу в надзорный орган (Роскомнадзор / местный DPA в ЕС).

## Передача третьим лицам

- Supabase (хостинг БД, EU)
- Google / Anthropic (обработка промпта — только на время запроса)
- ЮKassa / Stripe (оплата)
- Sentry (ошибки, без промптов)

## Cookies / аналитика

Десктоп-приложение не использует браузерные cookies. Сайт/лендинг — отдельная политика.

## Дети

Сервис не для лиц младше **16 лет**.
