# Discord / Telegram moderation

## Threats
- Scam bots DMing "free beats / crypto / fake support"
- Impersonation accounts (`PLG_Support`, `PluginFLP_Official`)
- Phishing links disguised as FL crack / sample packs

## Telegram
1. Official channel: `@pluginflp_updates` — **admins only post**
2. Support: `@pluginflp_support` — bot auto-reply with FAQ link + "never share password"
3. Enable **Slow mode** in groups if community chat opens
4. Report scam bots via Telegram abuse form
5. Pin: "Staff never DM first. Official site: pluginflp.app"

## Discord (when launched)
- Verification level: Medium
- AutoMod: block `@everyone`, invite links in first 24h
- Role `Support` only for staff with 2FA
- `#scam-reports` channel → mute + ban template
- Partner with [Discord Safety](https://discord.com/safety) for bot raids

## In-app
- Help links only to official URLs from `GET /v1/status` → `support` block
- No user-generated chat in v1 — reduces moderation surface

## Incident
If branded scam campaign: post on status page + Telegram within 1h.
